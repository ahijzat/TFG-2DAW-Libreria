# TFG

## Tematica
> Pagina que junta una libreria y una biblioteca

## Objetivos
> Es una aplicacion para poder comprar libros en la pagina web a traves de una cuenta donde
  se guardará los libros comprados y los libros sacados junto a las multas que tengas que pagar.

## Funcionalidades

### Roles
  - Admin: Podra acceder a todas las vistas y podra gestionar los libros disponibles ademas de añadir a cada cuenta la fecha limite para devolver el libro y las multas correspondientes.
  - Cliente: Podra acceder a los libros para comprarlos ademas de ver un historial con los libros comprados y sacados junto a la fecha limite y las multas que tiene que pagar.

### Vistas
  - Home
  - Busquedas
  - Login
  - Register
  - Nosotros
  - Contactanos
  - CRUD Admin: Libros, Usuarios, Configuracion
  - CRUD Perfil
